Discription:
Conole application that creates a recipe that contains ingredients and steps taken from the user.
The application is then capable of manipulating the recipe but scaling the quantities of the ingredients and reset it back
The application can also scrap the current recipe and create a new one.
The application can also filter and display recipes based on the ingredient, max colories or food group within the recipe.

Step By Step Instructions:

Start up the program in Visual Studio.
The UI application consists of 2 tabs:
- Create Recipe tab
- View Recipes tab

Create Recipe tab: 
User is require to enter the Name of the recipe with follow with the 
Ingredients name, amount used with measurement as well as the calories and foodgroup
After that follows the amount of steps and their discriptions.
After the ingredient is entered the user presses the corresponding button to save the ingredient.
After the step is entered the user presses the corresponding button to save the step.
After the buttons are pressed it displays the step/ingredient in the corresponding display.

Once the user wants to finalise the recipe the corresponding button should be pressed to store it in a list.
After the Recipe is created you are able to view in the Recipe Viewer tab.
When viewing recipes it will be displayed in alphabetical order.

View Recipes tab:
User can manipulate the recipes in 4 ways.
-View all recipes by clicking the view all button.
-Filter the recipes by selecting a filter then entering the details.
-Delete the name of the recipe entered from the list.
-Scale the recipe of the name entered.
