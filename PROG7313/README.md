#Discription :

 EZ Budget App is a personal budgeting application designed for adriod use only(currently).
 The application is used manage and track expenses of users whilst providing feedback and 
 helping them manage their person and/or shared budgets.


#Features :

 Login and signup - Users are able to create an account that is stored and then retrieved to use the application
 Creating expense categories - Users may create different categories for different expenses.
 Create expense entries - Users may create entries on the application to track any expenses made.
 Goal Setting - Users are able to set personal monthly goals and track their goals based on expenses made.
 View list of entries - Users are able to view all entries and are able to view these entries by category.
 Entry upload through files - Users may upload an entry through a document or picture and can be view too.
 Viewing spent amount - Users may seen at any given point how much they have spent based on their entries.


#Additional Features :

 Budget sharing - Multiple users may track each other's expenses through a shared budget plan.
 Multiple currency tracking - Users may track their expenses in different forms of currencies provided.


#Overview :

 Users register or signup before logging into the EZB application. After uses have logged in they may select
 to create a category for expenses. After expenses categories have been made users may create expense entries
 in their category that best suites the type of expenses and may do so by uploading a file or document. Users
 can the select the expense tracking and to track specific categories of expenses or expenses during a given 
 period of time. Users may nagivate to goals in-order to set personal goals where you may set a minimum or 
 maximum amount for your budgets per category which can be tracked here too. Users may navigate to shared 
 budgets where they will create a shared budget with another user/s where all entries may be created and track
 here. By using a list you may select the currency you would like to view your expenses in at any given time.
