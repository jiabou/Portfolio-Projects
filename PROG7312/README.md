# Intructions
  Main Menu [Form]
  -Consists of 3 buttons to navigate to 3 different forms namely Report Issue(Enabled), 
   Events/Announcements (Disabled) and Service Report Status(Disabled).

  Report Issue [Form]
  - User is required to enter a Location in the textbox.
  - User is required to select a Category in the dropdown box.
  - User is required to select a file using the select file button.
  - User is required to enter a Description of the report within the big text box.

  - As the user fills in the required information the progress bar on the right will fill up.

  - After the user has entered all the required information and clicked on the submit button
    a success message will appear and go back to the main menu or if there are any problems and
    error message relating to the issue will appear.

  - Click on the back button to go back to the main menu.
